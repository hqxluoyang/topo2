    开启图片选择框,无参

    需初始化组件模块后配置图片数组,配置参考样例

    Parameters:

    nametypeflagdescription
    [config] Object optional

    配置参数,无参数则只开启不设置

    nametypeflagdescription
    [images] Array optional

    窗口内应用的图片集合,不设置则沿用上次设置

    [isSet] Boolean optional

    确认是打开窗口还是设置窗口,为ture则不打开窗口只设置，false或不设则默认开启

    Returns:

    [Deferred]

    jquery的延迟执行函数

    Example:

     两种设置窗口内容的方式:
 1.instance.setComponent({
               windows: {
                   images: ["img/mo/eNodeB.png", "img/mo/eNodeB_1.png", "img/mo/eNodeB_2.png", "img/mo/eNodeB_3.png", "img/mo/eNodeB_4.png"]
               }
        });
   2.instance.open('imageSelect',{
           images: ["img/mo/eNodeB.png", "img/mo/eNodeB_1.png", "img/mo/eNodeB_2.png", "img/mo/eNodeB_3.png", "img/mo/eNodeB_4.png"],
           isSet:true
       });
 instance.open("imageSelect")
            .then(function(src){
               console.info(src);选中的图片
           });

    loading

    loading
    (
    • [config]
    )

    Defined in develop\component\windows\tools\loading.js:10

    进度条工具,常用于全局配置ajax执行时显示,ajax结束隐藏

    Parameters:

    nametypeflagdescription
    [config] Object optional

    配置参数,未配置时隐藏

    nametypeflagdescription
    [type] Number optional

    默认为0,暂时有2种,0和1

    [position] String optional

    出现在图上的位置,center,left,right,默认为right

    [logo] String optional

    type=0时，图中央的logo图标

    Example:

       $(document)
           .ajaxStart(function () {
                   instance.open("loading", {
                 type:0,
                   logo: "img/logo.png",
                   postion:"right"
               });
           })
           .ajaxError(function (e) {
                   console.error(e);
           })
           .ajaxStop(function () {
                   instance.open("loading");
           })

    进度条工具

    nametypeflagdescription
    [config] Object optional

    配置参数,未配置时不显示进度条

    nametypeflagdescription
    state Number

    当前进度,值域0-100(为100时，error未设或为false则延迟1秒后消失,error=true则延迟3秒后消失),未配置时不显示进度条

    styleSelect

    styleSelect
    (
    • [config]
    )
    Deferred

    Defined in develop\component\windows\tools\styleSelect.js:10

    公用的样式选择窗口

    Parameters:

    nametypeflagdescription
    [config] Object optional

    无值则全部展示 ,

    键对应窗口中样式的select框或Input框的name属性,值对应input内初始值或select的选中值 :

         支持属性有:
     width , height , fontSize , namePosition , borderWidth , borderRadius , direction , linkOffset , linkRadius , linkDash , linkGap ,
     arrowSize , arrowOffset , fontColor , color , borderColor

    Returns:

    [Deferred]

    jquery的延迟执行函数,回调函数传入的参数为,根据开启窗口的设置获取相应的值组成的对象

    Example:

     instance.open("styleSelect",{
               namePosition:"top",
               width:50,
               height:50,
               borderColor:"255,255,255"
   })
       .then(function(op){
           console.info(op);//若开启时指定namePosition,width,height,borderColor四项，则回调值也是这四项的
       });

    tips

    tips
    (
    • config
    )

    Defined in develop\component\windows\tools\tips.js:9

    鼠标移动提示框

    Parameters:

    nametypeflagdescription
    config Object

    配置参数

    nametypeflagdescription
    [show] Function optional

    未配置则认为要关闭该功能,展示面板内的内容,传入的参数为当前鼠标指向的元素

    [filter] Function optional

    指定何时显示,未配置则一直显示,传入的参数为当前鼠标指向的元素

    Example:

     两种配置方式,重复设置会替换上次设置
 1.instance.setComponent({
           windows: {
               tips: {
                   show: function (target) {
                       return "<div>名: " + target.val("name") + "</div>" + "<div> id: " + target.val("id") + "</div>"
                   },
                   filter: function (target) {
                       return target.getType() == QTopo.constant.NODE;
                   }
               }
           }
       });
   2.instance.open('tips',{
               show: function (target) {
                   return "<div>名: " + target.val("name") + "</div>" + "<div> id: " + target.val("id") + "</div>"
               },
               filter: function (target) {
                   return target.getType() == QTopo.constant.NODE;
               }
           });
